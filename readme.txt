== Description ==
Display client testimonials with slide previous and next features. And having different color settings from admin . 

<strong>Plugin Features</strong>

* Having color customization option in admin
* Shortcode System.
* Use this shortcode [wp-client-testimonial] where you want show the testimonial.
* Wordpress Custom Post Enabled.
* Easy to Use.
* You can use it in posts or pages.

== Installation ==
1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your WP-admin ->Client Testimonials,here we can add/manage client testimonials .
4. you can call client testimonial using shortcode
5. In page or post put shortcode directly 
` 
[wp-client-testimonial]
`
 In php template file call shortcode by 
`
<?php  echo do_shortcode("[wp-client-testimonial]"); ?>
`
